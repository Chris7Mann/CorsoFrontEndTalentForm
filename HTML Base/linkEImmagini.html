LINK E IMMAGINI

Link (tag a) - Sono di tipo inline

Attributo href che serve per indicare dove deve portare il link

  • Link che porta a una pagina interna al sito aprendola nella stessa scheda del browser

    Liste html Intestazioni html

  • Link che porta a una pagina interna al sito aprendola in una nuova scheda del browser (target="_blank")

    Esercizio Liste e formattazioni

  • Link che porta a una pagina esterna al sito

    N.B. generalmente i link esterni si fanno aprire in una nuova scheda del browser

    Sito di Raffaele Gaito

  • Link che ricarica la pagina (href="#")

    Ricarica pagina

  • Link che riporta all'inizio della pagina (href="#top")

    Torna all'inizio

  • Link bookmark o segnaposto (portano a una sezione di una pagina. href="#nomeId")

    Questo tipo di link è molto usato nei siti onepage

    Immaginiamo di volerci spostare alla sezione "link che porta a una pagina esterna al sito

    1. Aggiungo l'attributo id=nomeId alla sezione a cui voglio andare
    2. Nell'attributo href del link segnaposto metto #nomeId
    Sezione link esterni Ristoranti armarist Formattazioni non frequenti WebTea Progettazione

  • Link che permette di chiamare direttamente un numero telefonico (href="tel:numeroTelefonico")

    Chiama Fulvia

  • Link che apre il client di posta predefinito con il destinatario/i già impostato/i (href="Mailto: indirizzoEmail")

    Scrivici

  • Link che permette di iniziare una chat su whatsapp (href="https://wa.me/numeroTelefono")
